Projekt Pierwszy: Rozproszony System Mikroserwisów

Stworzyłem rozproszony system mikroserwisów z użyciem Docker, Kubernetes oraz Google Cloud Platform. System zarządza danymi użytkowników i zapewnia skalowalność.


// Przykładowy kod konfiguracji Kubernetes
apiVersion: apps/v1
kind: Deployment
metadata:
  name: user-service
spec:
  replicas: 3
  selector:
    matchLabels:
      app: user-service
  template:
    metadata:
      labels:
        app: user-service
    spec:
      containers:
      - name: user-service
        image: gcr.io/myproject/user-service:latest
        ports:
        - containerPort: 8080
---
apiVersion: v1
kind: Service
metadata:
  name: user-service
spec:
  selector:
    app: user-service
  ports:
    - protocol: TCP
      port: 80
      targetPort: 8080

Projekt Drugi: Aplikacja AI do Rozpoznawania Obrazów

Opracowałem aplikację AI do rozpoznawania obrazów z użyciem TensorFlow i PyTorch. Aplikacja analizuje obrazy i kategoryzuje je z wysoką dokładnością.


// Przykładowy kod TensorFlow
import tensorflow as tf
from tensorflow.keras import datasets, layers, models

# Załaduj i przetwórz dane
(train_images, train_labels), (test_images, test_labels) = datasets.cifar10.load_data()
train_images, test_images = train_images / 255.0, test_images / 255.0

# Zbuduj model
model = models.Sequential([
    layers.Conv2D(32, (3, 3), activation='relu', input_shape=(32, 32, 3)),
    layers.MaxPooling2D((2, 2)),
    layers.Conv2D(64, (3, 3), activation='relu'),
    layers.MaxPooling2D((2, 2)),
    layers.Conv2D(64, (3, 3), activation='relu'),
    layers.Flatten(),
    layers.Dense(64, activation='relu'),
    layers.Dense(10)
]);

model.compile(optimizer='adam',
              loss=tf.keras.losses.SparseCategoricalCrossentropy(from_logits=True),
              metrics=['accuracy']);

model.fit(train_images, train_labels, epochs=10, 
          validation_data=(test_images, test_labels));

Projekt Trzeci: Zaawansowany CMS

Opracowałem zaawansowany system zarządzania treścią (CMS) z użyciem Django, PostgreSQL oraz GraphQL. System umożliwia elastyczne zarządzanie treścią i integrację z różnymi usługami.


// Przykładowy kod Django + GraphQL
import graphene
from graphene_django.types import DjangoObjectType
from myapp.models import Article

class ArticleType(DjangoObjectType):
    class Meta:
        model = Article

class Query(graphene.ObjectType):
    all_articles = graphene.List(ArticleType)

    def resolve_all_articles(self, info, **kwargs):
        return Article.objects.all();

schema = graphene.Schema(query=Query);
